Shopping List Desktop App

Ver 1.0.0 Stable Release.

A small desktop app for creating a list of items to get on the internet or stuff to do during your work day.

- Use File category to add items.
- Double click list items to delete added elements.
- You can use Clear items statement in the file section to clear items list.
